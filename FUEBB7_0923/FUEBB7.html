git status - Változások listázása

git add - file hozzáadása

git commit - változások mentése

git push - Lokális változások feltöltése távoli repository-ba

git pull - Távoli repository változásainak letöltése és mergelése

git config - Git konfigurációs beállításainak megtekintésére, módosítására vagy globális beállítására szolgál

git clone - Távoli repository lemásolása local gépre. Automatikusan hozzádódik a távoli repository origin néven.
